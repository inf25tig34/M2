# Capítulo II: Interface com o utilizador

O website está organizado em 5 páginas. Existe uma página principal com informação geral sobre a banda Sétima Dominante, bem como quatro páginas dedicadas a diferentes aspetos do projeto musical.

Na página principal encontra-se uma secção de navegação, uma breve apresentação da banda, um pequeno resumo da sua história, uma lista com as principais secções do website e um formulário de interação para os visitantes.

A página Membros apresenta os elementos da banda, as suas funções e contributos para o projeto musical.

A página Influências Musicais descreve os artistas e bandas que inspiram a identidade sonora da Sétima Dominante, apresentando informações sobre cada uma dessas referências musicais.

A página Álbum Atonal é dedicada ao primeiro álbum da banda, contendo uma descrição do conceito geral do projeto, bem como informações sobre as sete músicas que o compõem e os temas abordados em cada uma delas.

A página Produção Musical apresenta o processo criativo e técnico utilizado pela banda, incluindo as diferentes etapas da criação de uma música, desde a composição até à masterização final. Esta página inclui ainda uma tabela com as principais fases da produção musical.

Todas as páginas possuem uma secção de navegação que permite ao utilizador deslocar-se facilmente entre os diferentes conteúdos do website.
